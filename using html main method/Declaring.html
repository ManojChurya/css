Methods are declared inside the body of a class, normally after the declaration of data fields.

		The general form (Syntax) of a method declaration is:		Example:
		modifier returntype methodname(formal-parameter-list)		public static void display(int a, float b) 
		{								{
			//method_body  							//method_body
		}								}
Methods declaration has five parts:
								Syntax			Example:
		1)Method modifiers			= modifier		-	public static
		2)Type of value the method returns	= returntype		-	void
		3)Name of the method			= methodname		-	display
		4)List of parameters			= formal-parameter-list	-	int a, float b
		5)Body of the method			= //method_body		-	//method_body
