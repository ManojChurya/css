Invoking of Main Method

  • Main Method

Once methods have been defined, they must be activated for operations.
The process of activating a method is know as invoking or calling.
The invoking is done using the dot operator as
objectname.methodname(actual-parameter-list);
objectname is the name of the object which we are calling the methodname.
The actual-parameter-list is a comma separated list of actual values that must match in type with the formal parameter list of method name declared in the class

	Here is an example of invoking of methods , method named cube, method class, MethodTest class is used to invoke the cube using dot operator.
			using System;
			class Method							|	//class containing the method
			{								|
				int cube (int x)					|	//define the cube method
				{							|	
					return ( x * x * x ) ;				|	
				}							|
			}								|	
			class MethodTest						|	//Client class to invoke the cube method
			{								|	
				public static void main()				|
				{							|
					Method M = new Method();			|	//Create object for invoking cube
					int y = M.Cube ( 5 ) ;				|	//Method Call
					Console.WriteLine( y );				|	The result is___
				}
			}
