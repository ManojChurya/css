Different types of language

Front-end

  • HTML(HyperText Markup Language)-to create pages
  • CSS(Cascading Style Sheets)-used to improve the presentation of a webpage
  • JS(Javascript)-helps to interact with the webpages.
  • jQuery-designed to simplify the client-side scripting of HTML.

Back-end

  • PHP(Hypertext Preprocessor)-easier for beginners to create websites
  • Ruby-provides a set of tools for accomplishing task
  • SQL(Structured Query Languag)-used in relational database servers
  • Python-high level language used for general purpose programming
  • ASP.Net( Active Server Pages)-using the C# and Visual Basic programming languages websites are built
  • Java-is a high level programming which are portable under different platform
